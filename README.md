🛡️ Top Notch Security Website
Top Notch Security is a professional security solutions and training platform designed to provide reliable protection services, expert consultancy, and industry-standard training programs. This project represents the official web presence of the company.
🚀 Features
🌐 Multi-page professional website
🛡️ Security services overview (physical & cyber)
🎓 Training & course access (online & physical)
👮 Hire security personnel feature
📩 Contact and inquiry forms
🖼️ Image-ready layout (customizable backgrounds)
📁 Project Structure

/project-folder
│── index.html      # Home page
│── about.html      # About the company
│── services.html   # Services offered
│── courses.html    # Training programs
│── hire.html       # Hire security guards
│── contact.html    # Contact page
│── logo.png        # Placeholder for images/logo
🛠️ Technologies Used
HTML5
CSS3
Google Fonts (Poppins)
🎯 Purpose
This platform is built to:
Showcase security services professionally
Provide access to training programs
Allow clients to request and hire security personnel
Establish a strong online presence for the brand
